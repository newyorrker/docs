## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

Что-бы загрузить экран нужно указывать соответствующий html файл applications.html или documents.html вместо index.html
